Paragraphs

This is my first real website! Paragraphs are made using the less than sign, a p, and the greater than sign, and closed using the same thing with a slash before the p.

Emphasis and bold

Em is for Emphasis and strong is for bold lettering.

Pro Tip:

Underlining can be confusing to users of websites, leading them to think the associated text is a link. Using bold and emphasis is a good alternative.

How to push from codespace to github:

  1. Type git status
  2. Type git add .
  3. Type git commit -m "commit comment goes here"
  4. Type git push

Fountdational Web Languages

HTML
HTML stands for Hyper Text Markup Language. It uses "tags" to annotate content.
CSS
CSS stands for Cascading Style Sheets. It is used for changing the presentation of a document written in a markup language.
Javascript
JavaScript is a high level programming language primarily known for making web pages interactive.
