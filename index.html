Thoughts on Progressive Enhancement

Progressive enhancement is an approach to web development that starts with the most important content and layers on more features depending on the tech used. This makes the essential information available to as many people as possible. This is important today because many people only have access to older technology. Some are only able to operate older technology, so it helps ensure that everyone is included.
